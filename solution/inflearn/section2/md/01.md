# [section1] 큰 수 출력하기
## 문제 요약 📚
- 자연수 N개의 정수 입력 받음
- 자신의 바로 앞 수보다 큰 수만 출력
- 첫 번째 수는 무조건 출력

## 소요 시간 ⌛️
- 약 1분

## 문제 풀이 📝
### Case1. Filter
```
function solution(arr) {
  return arr.filter( (cur, idx) => idx === 0 || cur > arr[idx -1] );
}

let arr=[7, 3, 9, 5, 6, 12];
console.log(solution(arr));
```
- 첫번째 수는 무조건 출력
  ```
  idx === 0
  ```
- 자신 바로 앞 수보다 큰 수만 한줄로 출력
  ```
  cur > arr[idx - 1]
  ```

## 실행속도 ⏱
```
s1: 약 0.035초
```
- 실행속도에 큰 차이가 없어서 가독성을 고려하면 좋을듯 하다
